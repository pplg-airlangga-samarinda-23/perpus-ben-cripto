# PEMINJAMAN BUKU DIGITAL

   **dekskripsi**
tujuan saya membuat web ini agar memudahkan registrasi perpustakaan dan mengecek buku, agar tidak ada kesalah pahaman terhadap sesama pengurus, dan yang bisa mengakses data hanya pengurus perpus, dan yang bisa mengisi link hanya peminjam saja.
   
   **fitur**
> login petugas dan peminjam
> pilih mengembalikan atau pinjam
> pilih buku yang di pinjan
> bayar denda ketikaa telat mengembalikan

   **tech stack/tools**
> Bahasa Pemrograman: PHP
> Database: MySQL
> Web Server: Apache (XAMPP)
> Text Editor: Visual Studio Code
